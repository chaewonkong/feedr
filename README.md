# feedr

RSS/Atom feed reader written in Rust.

## Features

- [ ] RSS/Atom feed parsing
- [ ] Periodic feed polling
- [ ] PostgreSQL storage
- [ ] REST API (axum)
- [ ] Web frontend

## Tech Stack

- **Runtime**: Tokio
- **HTTP Server**: Axum
- **HTTP Client**: Reqwest
- **Feed Parsing**: feed-rs
- **Database**: PostgreSQL (sqlx)
- **Serialization**: Serde

## Getting Started

```bash
cargo run
```
